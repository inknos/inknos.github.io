# inknos.github.io

This is my personal portfolio.

Static web page built with html, css, js

Feel free to use it!

## Running locally

Clone repo

`git clone https://github.com/inknos/inknos.gihub.io.git`
`cd inknos.github.io`

Install dependencies with npm

`npm install`
`gulp`

Run it locally

`npm start`
